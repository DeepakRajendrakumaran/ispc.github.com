Stmt* CreateForeachActiveStmt ( Symbol *  iterSym,
Stmt *  stmts,
SourcePos  pos 
)

This generates AST nodes for an __foreach_active statement. This construct can be synthesized ouf of the existing ForStmt (and other AST nodes), so here we just build up the AST that we need rather than having a new Stmt implementation for __foreach_active.

Parameters:
iterSymSymbol for the iteration variable (e.g. "i" in __foreach_active (i) { .. .}
stmtsStatements to execute each time through the loop, for each active program instance.
posPosition of the __foreach_active statement in the source file.

static bool lHasVaryingBreakOrContinue ( Stmt *  stmt) [static]

Given a statment, walk through it to see if there is a 'break' or 'continue' statement inside if its children, under varying control flow. We need to detect this case for loops since what might otherwise look like a 'uniform' loop needs to have code emitted to do all of the lane management stuff if this is the case.

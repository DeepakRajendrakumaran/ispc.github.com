00034 /** @file decl.h
00035     @brief Declarations related to type declarations; the parser basically
00036     creates instances of these classes, which are then turned into actual
00037     Types.
00038 
00039     Three classes work together to represent declarations.  As an example,
00040     consider a declaration like:
00041 
00042     static uniform int foo, bar[10];
00043 
00044     An instance of the Declaration class represents this entire declaration
00045     of two variables, 'foo' and 'bar'.  It holds a single instance of the
00046     DeclSpecs class represents the common specifiers for all of the
00047     variables--here, that the declaration has the 'static' and 'uniform'
00048     qualifiers, and that it's basic type is 'int'.  Then for each variable
00049     declaration, the Declaraiton class holds an instance of a Declarator,
00050     which in turn records the per-variable information like the name, array
00051     size (if any), initializer expression, etc.  
00052 */
